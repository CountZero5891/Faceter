Сгенерируйте два файла и закиньте на диск C и там же создайте папку temp
После того как вы перенесли файлы можете запускать программу
Программа сперва разбивает файлы на множество кусков объемом size_of_chunk
каждый из них сортируется и записывается во временный файл 
затем идет слияние отсортированных файлов в один файл. 
Файлы должны называться как BigFile0001.txt и BigFile0002.txt
